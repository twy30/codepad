This folder contains the "Reviewer" tools and tests.

# Review

Each file should have a `Version` property indicating the last review's
timestamp; i.e.

```YAML
Version: yyyy-MMM-dd HH:mm:ss
```

* [List](list.bash) files by their `Version` properties.
* [Spellcheck](spellcheck.bash) filenames and file contents.
* [Validate](validate.bash) file formats.
